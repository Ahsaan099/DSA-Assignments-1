Data Structures - Assignment 01
C++ implementations for the CS-III Data Structures assignment. This repository demonstrates foundational memory management, custom data structures, and basic algorithms.

🚀 Topics Covered
Pointers & References: Memory address manipulation.

Dynamic Arrays: Heap allocation and deallocation (new / delete[]).

Custom Array ADT: Insertion and searching within a custom class.

Search Algorithms: Iterative Linear and Binary Search.

📂 Files
Q6_Pointers.cpp: Reference and pointer operations.

Q7_DynamicArray.cpp: Runtime allocation, sum, and max calculation.

Q8_ArrayADT.cpp: Custom array class with search/insert methods.

Q9_Search.cpp: Linear and Binary search implementations.
